Transfer Credit Review Tool

An AI-built review pipeline for departmental transfer-credit evaluation. The tool turns an annual transfer review export into structured, scored, faculty-reviewable materials and a registrar-ready output.

Problem

Transfer-credit review requires departments to compare external course records against local catalog expectations, collect faculty input, preserve decisions, and return clean records for registrar processing.

Artifact Includes

  • Transfer-credit review skill
  • Parsed and deduplicated course records
  • Formatted department review workbook
  • Faculty reviewer Word documents
  • Annotation-tracked review workbook
  • Registrar-ready export

Process

  • Filter records by UST subject code.
  • Deduplicate to unique course and institution pairs.
  • Gather course descriptions, catalog links, and syllabus links.
  • Score candidate matches against the UST course catalog with TF-IDF cosine similarity.
  • Generate reviewer materials and extract annotations back into the workbook.
  • Produce a registrar-ready export without publishing private review decisions.

Result

Used for the 2026 Chemistry transfer-credit review with 109 reviewed courses after 5 were held for additional review.

Biochar Simulator

An AI-built Python package for generating model biochar surfaces for molecular dynamics simulations and visualization.

Problem

Building plausible biochar structures by hand is difficult, tedious, and error-prone, especially when researchers need structures that reflect production conditions or selected material properties.

Artifact Includes

  • Python package named biochar
  • Conda-forge install path
  • Generated molecular biochar structures
  • GROMACS-style simulation files
  • Structure statistics and property summaries
  • Sphinx documentation on Read the Docs

Process

  • Accept temperature/feedstock dependence or user-selected structural properties.
  • Generate biochar surfaces with configurable size, aromaticity, and heteroatom content.
  • Determine charges from the selected force field or a QM calculation.
  • Use RDKit and supporting Python libraries to build inspectable molecular structures.
  • Publish source, documentation, and installation instructions for reuse.

Result

Actively maintained research software used in Josh's biochar and PFAS-related computational chemistry work.
